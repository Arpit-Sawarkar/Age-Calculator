# Age Calculator

A simple Age Calculator desktop application built using **Python** and **Tkinter GUI**.
This application calculates the exact age in **years, months, and days** from the given Date of Birth.

This project was developed as part of my learning process. I also used AI tools like **ChatGPT** for guidance, suggestions, and improving the project structure and code quality.

---

## Features

* Calculate age from Date of Birth
* Shows years, months, and days
* Clean Tkinter GUI interface
* Input validation with error message
* Error handling for invalid date format

---

## Technologies Used

* Python
* Tkinter (GUI)
* Datetime module
